README Overview

This tool is a Python-based midi mapping tool

Requirements

Python 3.x
mido==1.3.2

Installation

Ensure Python 3.x is installed on your system.
Install the Requirements library using pip:

```bash
pip install Requirements.txt
```

Usage

Run the script from the command line:

```bash
python check-Inputs.py
```
